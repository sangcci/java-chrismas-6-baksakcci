# 기능 구현 목록
### 입력
- [ ] 날짜를 입력받는다
- [ ] 주문 목록을 입력받는다
- [ ] 빈 값이 오지 않았는지 확인한다
### `Order`
- [ ] 메뉴를 주문한다
  - [ ] 주문 메뉴가 중복되는지 확인한다
  - [ ] 존재하는 메뉴인지 확인한다
  - [ ] 메뉴가 20개가 넘는지 확인한다
  - [ ] 총 주문 가격이 1만원이 넘는지 확인한다
### `GiftBenefit`
- [ ] 샴페인을 증정한다
  - [ ] 할인 전 총 주문 금액이 12만원이 넘는지 확인한다
### `DiscountBenefit`
- [ ] 할인한다
  - [ ] 크리스마스 디데이 할인 정책을 적용한다
    - [ ] 할인 정책 적용 기간인지 확인한다
  - [ ] 일주일 할인 정책을 적용한다
    - [ ] 평일인지 주말인지 확인한다
  - [ ] 특별 할인 정책을 적용한다
    - [ ] 별이 있는 날짜인지 확인한다
- [ ] 증정 혜택이 있는지 확인한다
### `EventBadgeBenefit`
- [ ] 뱃지를 부여한다
  - [ ] 혜택 금액에 따라 차등 부여한다
